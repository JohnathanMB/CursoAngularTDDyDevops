TIPOS:
[I]	* string -> Cadena de texto 
[I]	* number -> Numero
[I]	* boolean -> tipo lógico buleano (True || false)
[I]	* array -> arreglo donde se ingresan datos de un tipo definido (primer posición = 0)
	* any -> Es un tipo de dato sin declarar, es decir, se usa cuando no se tiene la certeza de que tipo de dato se va a guardar en la variable 
	* duplas -> un arreglo de llave valor. Se debe definir el tipo de datos que habrá en cada casilla 


NOTA:
	** Se aconceja usar lo menor posible los datos Any
